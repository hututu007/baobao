# cool-breeze 凉风

#### 介绍
凉风有信，秋月无边，亏我思娇的情绪，好比那度日如年。

#### 软件架构
微信小程序原生开发加微云开发。


#### 目录说明

-  miniprogram
这是小程序的前端部分  
-  cloudfunctions
这是小程序的后端部分
-  project.config.json
这是云开发小程序的整个配置文件

#### miniprogram的目录划分

-  components
组件的目录
-  images
图片目录
-  models
这个目录打算放一些东西，但这个项目没用
-  pages
页面目录
-  style
样式目录，这个项目没用
-  util
工具目录
-  config.js
前端部分的配置文件
